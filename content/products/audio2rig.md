**Overview:**  
AUDIO2RIG is an interface device for audio and PTT control conversion between computers and radio stations, with audio transformer and optocoupler isolation, compatible and suitable for radio equipment such as Kenwu, Yaesu, ICOM, etc., supports VOXPTT control, audio uses USB sound card conversion interface, small size and easy installation. Supports Windows, Android, OS X operating systems.  
  
**Functions:**  
1. Integrated sound card, PTT control two-in-one;  
2. Supports VOXPTT control ();  
3. Mini design, size is only 53*24*14mm (excluding the protruding part of the USB head);  
4. Single 3.5mm plug 4PIN function definition, the definition can be changed by yourself, and different devices can be flexibly connected;  
5. Audio and PTT are physically isolated to reduce the problem of crosstalk between computers.  
  
**Package Include:**  
1 x USB2RIG Radio USB sound card connector
